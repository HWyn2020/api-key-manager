# Claude Code Skill

## What This Skill Does
Delegates build tasks to Claude Code CLI and returns the output.

## How to Use
Run the following bash command with your task as the argument:
```bash
/home/henry/workspace/skills/claude-code/run.sh "your full task prompt here"
```

## Rules
- Be specific — Claude Code only knows what you tell it
- Include target file paths in your prompt
- Review output before reporting to VJ
- Always follow with 4 Red Team protocol
